function definition:
    name:
        setWindowSize
    parameters:
        width : integer
        height : integer

    functionality:
        resizes the window to (width, height).
        (width, height) are in pixels.

function usage example:
    snippet 1:
        setWindowSize(1000, 800);

    snippet 2:
        integer height = 500;
        setWindowSize(2*height, height);
